# 義務教育5教科クイズ（25問）— GitHub Pages版

## 目的
小学校〜中学校範囲の5教科（国・数・英・理・社）からランダム出題し、
採点・解説・分析（レーダーチャート）と、端末内履歴（最近10回グラフ）を提供する。

## 公開形態
- GitHub Pages（静的ホスティング）
- 合言葉（パスフレーズ）による簡易アクセス制御
- 端末内保存（localStorage）で履歴を保持（サーバ不要）

## 主な機能
- 25問（各教科5問）をランダム出題
- 出題タグ：学年帯（小/中）・難易度（基礎/標準/発展）
- 難易度比率：3:4:3 をなるべく維持
- 25問内の重複禁止（問題かぶり防止）
- 似た出題パターンの連発を抑制
- 採点結果の表示（教科別内訳、時間）
- 問題番号クリックで解説表示
- レーダーチャート（教科別正答率）
- AI文章分析（正誤×時間の傾向から文章生成）
- 履歴（この端末）：
  - 最近10回の正答率推移（折れ線）
  - 全期間の教科別平均・全体平均
  - 履歴一覧表示・全削除

## 技術構成
- HTML / CSS / Vanilla JavaScript
- 問題バンク：bank.js（固定問題＋テンプレ生成）
- PWA：manifest.json + sw.js（オフライン対応・キャッシュ）

## データ保存
- ロック解除フラグ：localStorage
- 履歴（直近最大50回）：localStorage
※サーバ未使用、同一端末・同一ブラウザのみ保持

## 役割分担（現状）
- 企画・要件定義：ユーザー（プロダクトオーナー相当）
- 受入テスト・公開運用：ユーザー（QA/Release相当）
- 設計・実装：ChatGPT（開発者相当）
