# ロードマップ（改善計画）

## Phase 1：問題品質の強化（短期）
- 社会・理科の固定問題を入試レベルで増量（段階的に拡張）
- 誤答選択肢を「迷うが筋が通る」形に統一（倫理・事実破綻の排除）
- 国語・英語：短文読解／穴埋めの割合を増加

## Phase 2：分析の精緻化（中期）
- pattern別の弱点抽出を強化（出題数の母数条件を厳密化）
- “速い誤答”の種類（見落とし/語句誤認/条件読み違い）の分類を追加
- 教科別の改善提案テンプレ（次の学習行動）を自動生成

## Phase 3：同期・共有（将来案）
- ログインを導入し、複数端末で履歴統合
- クラス/塾単位の集計（匿名化）とダッシュボード
※コスト・運用負荷が上がるため、必要性を見て判断
